35  /**
36   * <p>
37   * Represent all Prolog number data type. Is an abstract class that contains all
38   * commons method related to number data types. In Prolog the number data types
39   * are Integer and Float. Some Prolog implementations have and extension for
40   * this legacy data types. They are Long (Large Integer Number) and Double
41   * (Double precision Floating Point Number). For Prolog implementations that no
42   * have support for Long and Double data types, they are implement this classes
43   * holding a Prolog integer for Long case and Prolog float for Double case.
44   * </p>
45   * <p>
46   * Two Prolog integers numbers are equals if and only if they are integers
47   * number and have the same value.Two Prolog integers numbers unify if are
48   * equals or at least one PrologTerm is a free variable Two Prolog floats
49   * numbers are equals if and only if they are floats number and have the same
50   * value.Two Prolog floats numbers unify if are equals or at least one
51   * PrologTerm is a free variable Two Prolog longs numbers are equals if and only
52   * if they are longs number and have the same value.Two Prolog longs numbers
53   * unify if are equals or at least one PrologTerm is a free variable Two Prolog
54   * doubles numbers are equals if and only if they are doubles number and have
55   * the same value.Two Prolog doubles numbers unify if are equals or at least one
56   * PrologTerm is a free variable
57   * </p>
58   * <p>
59   * Some Prolog implementations consider that integers and longs are equals if
60   * they have the same value and unify if have the same value or at least one
61   * PrologTerm is a free variable. Some Prolog implementations consider that
62   * floats and double are equals if they have the same value and unify if have
63   * the same value or at least one PrologTerm is a free variable.
64   * </p>
65   * 
66   * @author Jose Zalacain
67   * @since 1.0
68   */
